# Holberton School Web Front-End

This repository contains my work and practice projects for front-end development. The focus is on building fundamental skills using HTML, CSS, Bootstrap, and browser storage techniques.

# Overview

In this project, I explored different aspects of front-end development, including layout design, responsive pages, form handling, and client-side data storage. Each folder represents a specific topic or concept that I worked on.

# Project Structure
.vscode
Configuration files for my development environment.
Bootstrap
Exercises and examples using Bootstrap for styling and layout.
Cookies_local_storage
Practice with browser storage, including cookies and local storage.
form
Work related to HTML forms, validation, and user input handling.
images
Assets used across the project.
responsive_design
Examples of responsive layouts using CSS and media queries.
README.md
Project documentation.

# What I Learned

Through this project, I improved my understanding of:

Building structured web pages with HTML
Styling with CSS and Bootstrap
Making responsive designs for different screen sizes
Handling user input with forms
Storing data on the client side using cookies and local storage

# How to Run

You can open any of the HTML files directly in your browser. No additional setup is required.

git clone https://github.com/your-username/holbertonschool-web_front_end.git
cd holbertonschool-web_front_end

Then open the files with your browser.
